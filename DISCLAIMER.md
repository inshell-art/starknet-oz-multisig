# Disclaimer

This repository provides process templates for operational security (OPSEC) and agent-assisted onchain operations.

It is **not** a security guarantee, audit, or professional advice. You are responsible for:
- validating all transactions and code you deploy
- protecting your keys and signing environments
- testing thoroughly on testnets before mainnet
- understanding the risks of automation, LLM agents, and CI/CD systems

Use at your own risk.
