# Artifacts

This directory stores machine-readable outputs generated by the scripts.

## Layout

- `artifacts/<network>/multisig.class.json`
- `artifacts/<network>/multisig.<label>.json`
- `artifacts/<network>/counter.class.json`
- `artifacts/<network>/counter.json`
- `artifacts/<network>/rehearsal.counter.json`

`<network>` is one of: `devnet`, `sepolia`, `mainnet` (or any custom label you choose).
`<label>` comes from `--label` (or `MULTISIG_LABEL`) when deploying; default is `primary`.

## Example schemas

- `example.multisig.json` shows the per-instance multisig shape.
- `example.counter.json` shows the Counter deploy shape.

Real artifacts are ignored by git (see `artifacts/.gitignore`).
